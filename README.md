# dimensional

Interactive floor-plan furniture simulator. Upload a blueprint image, calibrate real-world scale, and place furniture shapes at true dimensions.

## Run

```bash
npm install
npm run dev
```

## Usage

1. Upload a PNG/JPG/WebP floor plan
2. Click **Calibrate**, then click two points on a known wall length and enter that length
3. Place furniture from the left catalog; drag, rotate, and edit sizes in the inspector
4. Toggle **ft / in** or **metric** anytime — sizes convert live

Layout state is saved in `localStorage`.
